Postgres commands
    psql - log you in to the CLI
        psql DATABASENAME USERNAME
    \q - close the CLI
    \l - lists all databases
    \du - lists all users
    \c databaseName - connects to databaseName
    \dt - list tables in a database
    \d tableName - details of a table

Steps when starting a new project with a DB:
1) Create a user (Choose a user)
2) Create the database
3) Create tables
4) Insert data into tables

1) Create a user
    CREATE USER username WITH CREATEDB PASSWORD 'password';
2) Create a database
    CREATE DATABASE databasename WITH OWNER username;
3) Create tables
    CREATE TABLE tableName(s) (
        id SERIAL PRIMARY KEY,
        colName dataType optionalConstraints
    );

    To run a sql file:
        psql -d boardgame_dev < sql_code.sql
