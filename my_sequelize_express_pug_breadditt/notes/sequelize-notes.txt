Step 1: Initialize npm
    npm init -y

Step 2: Install node packages
    sequelize@^5.0.0
    sequelize-cli@^5.0.0
    pg
        npm uninstall pg
        npm install pg

Step 3: Initialize sequelize
    npx sequelize init

Step 4: Edit the config file
    change:
        username
        password
        database
        dialect
    remove:
        operatorsAliases
    add:
        seederStorage: sequelize

Step 5: Create database
    npx sequelize db:create

Step 6: Create tables
    npx sequelize model:generate --name User --attributes name:string,col2:interger,col3:boolean
        creates the table model
        model names: capital and singular
        table/migration names: capital and plural
    npx sequelize db:migrate

    npx sequelize db:migrate:undo:all


Step 7: Insert data
    npx sequelize seed:generate --name userSeeds
    npx sequelize db:seed:all
    npx sequelize db:seed:undo:all

When fixing migration errors
    rollback seeds  npx sequelize db:seed:undo:all
    rollback migrations  npx sequelize db:migrate:undo:all
    re-migrate  npx sequelize db:migrate
    re-seed  npx sequelize db:seed:all

Associations
    hasOne()   one to one
    hasMany(), belongsTo()    one to many
    belongsToMany()  many to many

Query functions:
    build
    save
    create
    findByPk
    findAll
    destroy
